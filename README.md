# scalable-data-pipeline

- **Extract:** Reads raw data with minimal validation
- **Transform:** Applies explicit cleaning (dates, numeric casting, missing values)
- **Load:** Writes processed outputs (to be added)

## How to Run
1. Install dependencies:
   - `python3 -m pip install -r requirements.txt`

2. Run the pipeline:
   - `python3 main.py`

3. Output:
   - Processed data is written to `data/processed/orders_processed.csv`
